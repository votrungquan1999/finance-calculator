## Overview

This document provides detailed examples and templates for all UI copy elements, including tooltips, help text, form labels, and status messages. Use these as reference templates when creating new content.

## Tooltips & Help Text

### Concise Guidelines

**Tooltip Length**:
- 1-2 sentences maximum
- 10-20 words ideal
- Clear and actionable
- Avoid jargon

**Help Text Length**:
- 2-3 sentences maximum
- 20-40 words ideal
- Explain purpose and usage
- Include examples when helpful

### Tooltip Format

**Standard Format**:
"[Field Name]: [Brief explanation]. Example: [Sample value]"

**Examples**:
- "Interest Rate: Annual percentage rate for your loan. Example: 6.5 for 6.5%"
- "Loan Amount: Total amount you're borrowing. Example: 300000 for $300,000"
- "Loan Term: Duration of your loan in years. Example: 30 for 30 years"

### Tooltip Examples

## Accessibility Guidelines

### Screen Reader Support

**ARIA Labels**:
- Use descriptive labels for all interactive elements
- Provide context for form fields
- Include instructions for complex interactions
- Ensure all content is accessible

**Keyboard Navigation**:
- All interactive elements should be keyboard accessible
- Provide clear focus indicators
- Use logical tab order
- Include keyboard shortcuts where helpful

### Visual Accessibility

**Color Contrast**:
- Ensure sufficient contrast for all text
- Don't rely on color alone to convey information
- Provide alternative indicators for color-coded information
- Test with colorblind users

**Text Readability**:
- Use clear, readable fonts
- Provide adequate font sizes
- Use sufficient line spacing
- Avoid complex text formatting
